The following functions are available globally.

  • unreachable()

    An unreachable code path.

    This can be used for whenever the compiler can’t determine that a path is unreachable, such as dynamically terminating an iterator.

  • assertUnreachable(_:file:line:)

    Asserts that the code path is unreachable.

    Calls assertionFailure(_:file:line:) in unoptimized builds and unreachable() otherwise.

    Declaration

    Swift

    @inline(__always)
public func assertUnreachable(_ message: @autoclosure () -> String = "Encountered unreachable path",
                              file: StaticString = #file,
                              line: UInt = #line) -> Never

    Parameters

    message

    The message to print. The default is Encountered unreachable path.
    file

    The file name to print with the message. The default is the file where this function is called.
    line

    The line number to print with the message. The default is the line where this function is called.
