stopAndAccumulate

fun stopAndAccumulate(timerId: Any): Unit

Stop tracking time for the provided metric and associated object. Add a count to the corresponding bucket in the timing distribution. This will record an error if no start was called.

Parameters

timerId - The object to associate with this timing. This allows for concurrent timing of events associated with different objects to the same timespan metric.
