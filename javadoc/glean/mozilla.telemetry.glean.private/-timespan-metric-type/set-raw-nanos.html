setRawNanos

fun setRawNanos(elapsedNanos: Long): Unit

Explicitly set the timespan value, in nanoseconds.

This API should only be used if your library or application requires recording times in a way that can not make use of start/stop/cancel.

Care should be taken using this if the ping lifetime might contain more than one timespan measurement. To be safe, setRawNanos should generally be followed by sending a custom ping containing the timespan.

Parameters

elapsedNanos - The elapsed time to record, in nanoseconds.
