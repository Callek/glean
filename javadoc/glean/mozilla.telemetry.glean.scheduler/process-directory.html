processDirectory

fun processDirectory(lock: Any, storageDirectory: File): Boolean

Function to deserialize and process all serialized ping files. This function will ignore files that don't match the UUID regex and just delete them to prevent files from polluting the ping storage directory.

Return
Boolean representing the success of the upload task. This may be the value bubbled up from the callback, or if there was an error reading the files.
